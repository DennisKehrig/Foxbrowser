Firefox Home for iPad

Foxbrowser is an attempt to update Mozilla's Firefox Home iOS app with an iPad UI

This project is a fork of Firefox Home, you may recognize some Interface elements from the original app.

Sync support

Currently there is support for syncing your bookmarks, history and tabs to your iPad. Due to limitations in the Mozilla Weave sync provider, there is no support for saving bookmarks. This will be fixed in the future.
